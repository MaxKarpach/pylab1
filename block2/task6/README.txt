Пример Пузырьковой сортировки на массиве чисел.
На вход подаётся текстовый документ input.txt из двух строк: в первой строке размер массива, а во второй числа, разделённые пробелом.
Необходимо отсортировать массив и положить результат в файл output.txt.

Инструкция:
1) Вручную создаётся файл input.txt в соответствии с условием или запускается программа generator.py.
2) Запускается файл main.py.
3) Результат Пузырьковой сортировки input.txt помещается в output.txt.
